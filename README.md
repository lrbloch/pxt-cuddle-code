# pxt-cuddlecode ![Build status badge](https://github.com//workflows/MakeCode/badge.svg)

A customizable and programmable stuffed animal for girls ages 6-10 to build and program in their homes with adult collaboration and support

## Use this extension

This repository can be added as an **extension** in MakeCode.

* open https://makecode.microbit.org/
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for the URL of this repository and import

## Edit this extension

To edit this repository in MakeCode.

* open https://makecode.microbit.org/
* click on **Import** then click on **Import URL**
* paste the repository URL and click import
